\author{Emanuele Fontana}
\date{}


\begin{document}

\maketitle

\begin{abstract} This project focuses on the application of Green AI techniques for molecular property prediction. Following the CRISP-DM methodology, the work explores an adaptive Early Stopping mechanism based on Exponential Moving Average \cite{EMARecSys} (under review) for FineTuning Transformer-based models and Graph Neural Networks. The main goal is to analyze the trade-off between predictive accuracy and energy consumption with respect to a classic FineTuning Early Stopping method, showing that it is possible to achieve significant reductions in CO$_2$ emissions with minimal impact on performance. \end{abstract}
